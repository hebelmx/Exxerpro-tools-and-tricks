# Exxer Tools & Tricks

A curated collection of **Windows tools, tricks, and fixes** — distilled from real
workstation maintenance, written to be copy-pasteable and safe to reuse on any machine.

Every page here is **anonymized by default**: no machine-specific paths, hostnames, or
personal data. If something looks generic, that's on purpose — replace placeholders like
`<repo>` or `C:\Users\<you>` with your own.
